# 한자 공부 웹사이트 (Hanja Study Web)

중학교 교육용 기초 한자 900자를 포함하여, 고등학교 수준까지 확장 가능한 한자 학습 웹사이트입니다.

## 주요 기능
- **한자 카드 그리드**: 가나다순으로 정리된 한자 목록
- **획순 애니메이션**: HanziWriter 라이브러리를 활용한 실시간 획순 애니메이션
- **연습 모드**: 사용자가 직접 한자를 그려보며 학습하는 퀴즈 기능
- **대시보드**: 학습 진행 상황 확인
- **프리미엄 디자인**: 다크 모드와 골드 포인트, 글래스모피즘이 적용된 세련된 인터페이스

## 기술 스택
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- [HanziWriter](https://chanind.github.io/hanzi-writer/) 라이브러리

## 설치 및 실행
1. 이 저장소를 클론합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
